# Public-Shogi-Report

将棋ウォーズの棋譜を自動取得・Gemini AI 解析し、WebUI で閲覧・深掘り分析を行うシステム。

---

## ⚠️ 利用に関する重要な警告

- **本ツールは、必ずしもあなたの棋力向上や昇段を約束するものではありません。**
- **`.env` ファイルは絶対に Git にコミット・公開しないでください**（`.gitignore` で除外済み）
- **GitHub リポジトリは必ず Private に設定してください**
- **Gemini API キー・GitHub Personal Access Token を他人に知らせないでください**

本ツールは無償で提供されており、末尾の免責事項を必ず確認した上で、利用者自身の責任において使用してください。
本ツールの利用により生じた損害について、開発者は一切の責任を負いません。
APIキーや個人情報の管理は利用者自身が行ってください。
本ツールを利用した時点で、利用者は本免責事項に同意したものとみなします。
